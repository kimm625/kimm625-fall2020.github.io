Type Your Own Christmas!
2020
This website was created to work as a toy. I intended to make it feel like winter, thus I made a Christmas tree with coding. If you start typing in the text box at the bottom left, the tree gradually shows up. When the tree is complete, the light on the tree flashes and you can click anywhere on the screen to go back to the beginning. Enjoy typing your own Christmas!
