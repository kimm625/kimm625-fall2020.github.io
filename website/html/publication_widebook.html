Experimental Typography
(10.5 x 15.5 inch)
2019
The booklet focuses only on typography rather than image or content. This shows how words ‘wide’ and ‘narrow’ express contrast in one spread with only a typography layout. Each page is not unnatural because it is linked in some way, such as color, layout, and detailed elements. The project was designed and experimented more freely away from stereotypes.
